# temporalFilter
This project performs Gaussian smoothing common video files (mp4, avi, mkv, etc.) across spatial and temporal dimensions.

## Features
- CLI interface for ease of use
- Support for variety of video formats

## Dependencies
- FFmpeg must be installed to extract and reconstruct frames

---

# Running the Program
## Compile
- Run the make file to compile all files:
```
make
```

## Clean Executable and Temporary Files
- The following deletes the executable and other temporary files generated during execution:
```
make clean
```
